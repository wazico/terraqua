# TERRAQUA NATURE

Site web pour TERRAQUA NATURE - Mobilier d'aquarium & créations murales

## Installation

```bash
npm install
```

## Développement

```bash
npm run dev
```

Le site sera accessible sur `http://localhost:5173`

## Build

```bash
npm run build
```

## Technologies utilisées

- React 18
- Vite
- CSS3

## Structure du projet

- `src/components/` - Composants React
- `src/App.jsx` - Composant principal
- `src/index.css` - Styles globaux
